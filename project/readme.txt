Библтотеки:
sqlite - Работа с базой данных. Хранит информацию об аудиториях, бронированиях, жалобах, статистике
datatime - Работа с датами. Определяет сегодняшнюю дату, сравнивает с датами бронирований
random - Генерация случайных чисел. Используется только для создания демо-данных при первом запуске
flask (pip install Flask) - Веб-фреймворк. Создаёт сервер, обрабатывает маршруты (URL), рендерит HTML-шаблоны, принимает и обрабатывает GET/POST запросы
