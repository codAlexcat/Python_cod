Библтотеки:
sqlite - Работа с базой данных. Хранит информацию об аудиториях, бронированиях, жалобах, статистике
datatime - Работа с датами. Определяет сегодняшнюю дату, сравнивает с датами бронирований
random - Генерация случайных чисел. Используется только для создания демо-данных при первом запуске
flask (pip install Flask) - Веб-фреймворк. Создаёт сервер, обрабатывает маршруты (URL), рендерит HTML-шаблоны, принимает и обрабатывает GET/POST запросы

Функции из Flask:
request - Получает данные, которые пользователь отправил через форму
render_template - Отображает HTML-шаблон. Превращает файлы из папки templates/ в готовую HTML-страницу и передаёт в неё данные
redirect - Перенаправляет пользователя на другую страницу (например, после бронирования возвращает на список аудиторий)
url_for - Генерирует URL по имени функции. Например, url_for('rooms') превращается в /rooms. Удобно, чтобы не писать адреса вручную
